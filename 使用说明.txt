第一步：获取免费的API Key（3分钟）
====================================================

1. 打开 https://platform.deepseek.com
   （不需要VPN，国内直接访问）

2. 点击「注册」→ 用手机号或邮箱注册

3. 登录后，点击左侧菜单「API Keys」

4. 点击「创建API Key」
   - 取个名字，比如「升学咨询」
   - 复制生成的Key（以 sk- 开头）

5. 这个Key就是你的AI接口钥匙
   ✅ 注册送500万token，够你用很久
   ✅ 每次对话大约消耗几百token，可以用很久很久

====================================================
第二步：运行页面（马上就能用）
====================================================

方法一：直接打开（最简单）
   双击 index.html 文件，在浏览器中打开
   → 点击右上角「⚙️ 设置」
   → 粘贴你的API Key
   → 开始咨询！

方法二：分享给家长（需要发布到网上）
   方式A - GitHub Pages（推荐，免费）：
     1. 注册 github.com
     2. 新建仓库 → 上传 index.html
     3. 开启 GitHub Pages → 获得公开链接
     4. 把链接发给家长
   
   方式B - 用服务器临时共享：
     在终端执行以下命令（macOS/Linux）：
     cd /Users/coeyliu/Documents/Codex/2026-06-25/hi/outputs/升学AI助手
     python3 -m http.server 8080
     然后同一局域网内的人可以访问 http://你电脑IP:8080

   方式C - 直接用网盘分享：
     把 index.html 文件发给家长
     家长下载后用浏览器打开即可
     （⚠️ 但需要家长也设置自己的API Key）

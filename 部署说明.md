# 升学AI助手 — 部署说明

## 需要准备的东西

1. **XorPay 账号**（审核中，通过后获得 APP ID + APP SECRET）
2. **Cloudflare 账号**（免费，cloudflare.com 注册）
3. **你自己的 DeepSeek API Key**
4. **页面部署方式**：GitHub Pages / Vercel / 任意静态托管

---

## 第一步：部署 Cloudflare Workers

### 1.1 注册 Cloudflare

1. 打开 [cloudflare.com](https://cloudflare.com) → 注册免费账号
2. 验证邮箱后登录

### 1.2 创建 KV 命名空间

1. 左侧菜单 → **Workers & Pages** → **KV**
2. 点击 **创建命名空间**
3. 名称填写：`PAYMENTS`
4. 创建完成

### 1.3 创建 Worker

1. 左侧菜单 → **Workers & Pages** → **创建 Worker**
2. 名称填写：`shengxue-pay`（或你喜欢的名字）
3. 点击 **编辑代码**
4. 删除默认代码，粘贴 `workers/pay.js` 的内容
5. 点击 **保存并部署**

### 1.4 绑定 KV 到 Worker

1. 在 Worker 详情页 → **设置** → **变量**
2. 找到 **KV 命名空间绑定** → **添加绑定**
3. 变量名称：`PAYMENTS`（代码里用这个名）
4. KV 命名空间：选择刚才创建的 `PAYMENTS`
5. 点击 **保存**

### 1.5 设置环境变量

在 Worker → **设置** → **变量** → **环境变量**：

| 变量名 | 值 |
|-------|---|
| `XORPAY_APP_ID` | 审核通过后 XorPay 后台的 APP ID |
| `XORPAY_APP_SECRET` | XorPay 后台的 APP SECRET |
| `XORPAY_PAY_TYPE` | `wxpay`（或 `alipay`） |
| `ALLOWED_ORIGINS` | 你的页面域名，如 `https://xxx.github.io` |

### 1.6 配置 XorPay 回调

1. 在 XorPay 后台 → **应用管理** → **支付回调地址**
2. 填写：`https://shengxue-pay.你的子域名.workers.dev/pay/callback`
3. 保存

### 1.7 测试 Worker

访问：`https://shengxue-pay.你的子域名.workers.dev`
看到 `{"error":"Not Found"}` 就说明部署成功了。

## 第二步：更新前端页面

### 2.1 在 `index.html` 中填入 Worker 地址

搜索 `WORKER_URL`，替换为你的 Worker 地址：
```
https://shengxue-pay.你的子域名.workers.dev
```

### 2.2 部署前端页面

**方式一：GitHub Pages（推荐，免费）**

1. 注册 [github.com](https://github.com) → 创建仓库
2. 把 `升学AI助手/` 文件夹里的所有文件上传到仓库
3. 仓库 → **Settings** → **Pages**
4. Source 选 **Deploy from a branch** → 选 `main` → `/` (root)
5. 等 2 分钟 → 你的页面就在 `https://你的用户名.github.io` 了

**方式二：Vercel（更快）**

1. 注册 [vercel.com](https://vercel.com) → 用 GitHub 登录
2. 导入仓库 → 部署 → 搞定
3. 比 GitHub Pages 多一个自定义域名更简单

## 第三步的替代方案：全自动支付

如果用户量上来后不想手动发激活链接，就走 Worker 全自动支付：

1. 用户在前端点「升级」
2. 前端调 Worker API 创建订单，返回二维码
3. 用户扫码付款
4. XorPay 回调 Worker
5. Worker 更新 KV 中的 paid_amount
6. 前端轮询到已付款 → 自动解锁

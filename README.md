## 快速开始

### 1. 获取 DeepSeek API Key

1. 打开 https://platform.deepseek.com（国内直接访问）
2. 注册账号（手机号或邮箱）
3. 登录后，进入左侧菜单「API Keys」
4. 点击「创建API Key」，取名后复制生成的 `sk-` 开头的 Key

> 注册送 500 万 token 额度，足够长期使用。

### 2. 运行页面

**本地打开（最简单）**

双击 `index.html` 在浏览器打开 → 点击右上角「⚙️ 设置」→ 粘贴 API Key → 开始咨询！

**发布到网上（分享给家长）**

推荐用 GitHub Pages（免费）：
1. 注册 github.com
2. 新建仓库 → 上传 `index.html`
3. 开启 GitHub Pages → 获得公开链接
4. 把链接发给家长

也可以用 `python3 -m http.server 8080` 临时共享（局域网），或直接把文件发给家长（对方也需要设置自己的 API Key）。
